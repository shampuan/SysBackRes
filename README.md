# SysBackRes

This is an System Backup and Rescue Tool for Debian-Linux Systems.

This wizard is designed to facilitate your system backup, disk cloning, image creation, image recovery and backup verification operations. The program provides reliable and fast operations using the high-performance ddrescue backend tool.

Languages: Azerbaycan, German, English (Default), French, Italian, Japanese, Korean, Russian, Turkish
